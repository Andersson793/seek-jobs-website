# seek-jobs-open-source

## Como rodar

1. clone o repositório:

```bash
git clone https://github.com/LukasdeSouza/seek-jobs-website.git
```
2. acesse o diretório raiz:
```bash
cd seek-jobs-website
```
3. instale as dependências:
```bash
yarn install
```
4. inicie o servidor:
```bash
yarn start
```
